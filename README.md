# 🛍️ Mall Customer Data Analysis

This project focuses on analyzing customer data from a shopping mall using **Python data analysis and visualization libraries**. The goal is to explore customer behavior, identify patterns, and gain insights into spending habits.

## 📌 Project Overview

The dataset contains customer details such as:

* Gender
* Age
* Annual Income
* Spending Score

Using this data, we perform:

* Data cleaning and preprocessing
* Statistical analysis
* Group-based insights
* Data visualization

---

## 🛠️ Technologies Used

* **NumPy** – for numerical operations
* **Pandas** – for data manipulation
* **Matplotlib** – for basic visualizations
* **Seaborn** – for advanced visualizations

---

## 📊 Key Analysis Performed

* Checked dataset structure and summary statistics
* Identified missing values
* Converted categorical data (Gender → Numeric)
* Calculated:

  * Average age of customers
  * Average annual income
* Grouped data to find:

  * Average spending score by gender
  * Average income by gender

---

## 📈 Visualizations

The project includes:

* Count plot of gender distribution
* Age distribution histogram with KDE
* Boxplot showing gender vs spending score

These visualizations help understand customer segmentation and behavior patterns.

---

## 📂 Dataset

* File used: `Mall_Customers.csv`
* Contains structured customer data for analysis

## 🎯 Insights

* Gender-based spending patterns can be observed
* Income and age distributions reveal customer segments
* Useful for marketing and business decision-making
